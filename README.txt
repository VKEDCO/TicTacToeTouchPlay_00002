1) Eclipse Project (TicTacToeTouchPlay_00002.rar) 
with an Android application that implements a Tic-Tac-Toe game
application for the Tic-Tac-Toe content provider implemented
in the TicTacToeCPCreator_00001 project (https://github.com/VKEDCO/TicTacToeGameCPCreator_00001).

2) This game application retrieves optimal min/max moves for
given board states from the content provider.

3) Targets: Android 2.3+, Eclipse Helios 3.6

4) To run: 
   4.1) install and register the content provider;
   4.2) unzip TicTacToeTouchPlay_00002.rar into a folder; 
   4.3) open in Eclipse; 
   4.4) Run as Android application
